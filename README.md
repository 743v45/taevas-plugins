## 可用插件

| 插件 | 描述 | 文档 |
|------|------|------|
| **sound-hooks** | 任务事件声音通知 | [plugins/sound-hooks](plugins/sound-hooks) |

## 快速开始

```bash
# 添加市场
claude plugin marketplace add 743v45/taevas-plugins

# 安装插件
claude plugin install sound-hooks@taevas-plugins

# 更新插件
claude plugin marketplace update taevas-plugins
claude plugin update sound-hooks@taevas-plugins
```
